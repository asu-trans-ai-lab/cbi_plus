# DATA_FORMATS — every dataset in this repo, for notebooks

Generated by `python scripts/build_data_pack.py`. Each dataset has a
60-row `_sample.csv`, a zstd `.parquet` (tracked when <= 10 MB —
`.parquet.localonly` means rerun the script to regenerate), and a
`.geojson` where the data carries geometry. Load pattern:

```python
import pandas as pd
df = pd.read_parquet('benchmarks/_datapack/pems_i10_2018_states.parquet')
```
